# Valentine Dhamaka

## Overview
Valentine Dhamaka is a fun and engaging application designed to celebrate love and friendship. This README provides comprehensive setup instructions for running the application on both PC and phone platforms.

## Prerequisites
Before you begin, ensure you have the following prerequisites:
- **Node.js**: Install [Node.js](https://nodejs.org/) (LTS version recommended).
- **npm**: Comes pre-installed with Node.js.
- **Git**: Make sure Git is installed to clone the repository.
- **Android Studio** (for Android development) or **Xcode** (for iOS development).

## Installation Steps
### Running on PC
1. **Clone the Repository**
   ```bash
   git clone https://github.com/DancingRaja/valentine-dhamaka.git
   cd valentine-dhamaka
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Start the Local Server**
   ```bash
   npm start
   ```
   Your application should now be running on [http://localhost:3000](http://localhost:3000).

### Running on Phone
#### For Android
1. **Clone the Repository**
   ```bash
   git clone https://github.com/DancingRaja/valentine-dhamaka.git
   cd valentine-dhamaka
   ```

2. **Install Android Studio** and set up the Android SDK.

3. **Open Android Studio**
   - Import the project in Android Studio.

4. **Run the Application**
   - Connect your Android device or start an emulator.
   - Click on 'Run' in Android Studio.

#### For iOS
1. **Clone the Repository**
   ```bash
   git clone https://github.com/DancingRaja/valentine-dhamaka.git
   cd valentine-dhamaka
   ```

2. **Install Xcode**
   - Make sure you have Xcode installed from the App Store.

3. **Open the Xcode Project**
   - Open the `.xcworkspace` file in Xcode.

4. **Run the Application**
   - Choose a simulator or connect your iOS device.
   - Click the 'Run' button in Xcode.

## Troubleshooting Guides
- **Common Issues**
  - If the application fails to start, ensure that all dependencies are correctly installed.
  - Verify your Node.js and npm versions using `node -v` and `npm -v` commands.

- **Building Errors**
  - Check for typos in configuration files and ensure all paths are correctly set.

- **Network Issues**
  - Ensure you are connected to the internet, and the server URL is reachable.
